1. Introduction
=============================================================================

This media contains exported patient image data generated by ViewForum.
This data can be viewed using the Philips DICOM Viewer R1.1 application that is
present on this medium.

=============================================================================
2. Main contents on this medium
=============================================================================

    1. ViewForum generated patient data
    2. Philips DICOM Viewer R1.1
    3. An INDEX.HTM page to launch Philips DICOM Viewer R1.1
    4. This README.TXT

=============================================================================
3. Minimum Requirements
=============================================================================

    1) Windows XP
    2) Microsoft Windows Explorer version 5.0 or higher

=============================================================================
4. Launching instructions
=============================================================================

    1) Insert this medium in the CD/DVD reader drive of your system.
    2) To launch Philips DICOM Viewer R1.1, double-click on the INDEX.HTM file
       present on this medium.
    3) A warning for the usage of an Active X object may 
       (depending on your Internet Explorer settings) be displayed.
       Click Yes on the warning message window. This use of this
       object is necessary to launch Philips DICOM Viewer R1.1 successfully.
    4) Click on the displayed link to start Philips DICOM Viewer R1.1.
    5) Philips DICOM Viewer R1.1 will then be launched.
