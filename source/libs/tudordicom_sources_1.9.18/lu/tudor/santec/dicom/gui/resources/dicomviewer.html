The Tudor DICOM Viewer

The Tudor DICOM Viewer is an application to view all kinds of DICOM images. Several series of single or multi slice DICOM images can be loaded into the program from the file system, the local DICOM STORE, a DICOM CD or DICOMDIR and a PACS using DICOM C-Find. The screen of the viewer is split into 4 parts.

  1. The View area in the center of the window shows the selected image. The area can be splitted in 4 different modes. One image only, to vertical, two horizontal or 4 images in parallel. The focus, which will be shown as a red border, can be set to an image by clicking it. Image operations and loading images from the Series bar are always applied to the focused image.

  2. The Tool bar on top, allows the user to open images, choose how to split the screen and access the configuration and help dialog.

  3. The Series bar on the right shows all open DICOM series with some additional information. Images can be loaded into the selected View area by clicking a series. Series can be closed by left-clicking a series.

  4. The Tool bar on the left allows the user to choose an operation mode or action on the selected image.

The following mouse-key combinations can be used to control the Viewer.

  • Use Left Mouse button to apply the Mode selected in the Tool bar.

  • Use Left Mouse button + CTRL to Move the Image.

  • Use Right Mouse button to show the context menu.

  • Use the Mouse wheel to run trough Slices.

  • Use the Mouse wheel + CTRL to zoom the Image.

  • Use the SHIFT key to accelerate all modes.
